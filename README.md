# MT4交易监控系统

一个实时监控MT4交易账户的系统，包括前端界面和后端数据处理。

## 功能特点

- 实时显示MT4账户余额、权益、保证金等信息
- 可视化账户盈亏趋势
- 支持多账户监控
- 自动定时刷新数据
- 响应式设计，支持移动设备

## 项目结构
mt4-trading-system/
├── client/                 # 前端代码 (已部署在Netlify)
├── server/                 # 后端API服务
├── mt4-collector/          # MT4数据收集器
└── README.md
